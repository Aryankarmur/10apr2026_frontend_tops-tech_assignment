# 🎬 Movie Watchlist

A Movie Watchlist application built using **React + Vite + Firebase Firestore**.

---

## Features

- Add Movie
- Edit Movie
- Delete Movie
- Firebase Firestore Database
- React Router
- React.lazy() Code Splitting
- Responsive Design
- Environment Variables
- Ready for Netlify/Firebase Hosting

---

## Technologies

- React
- Vite
- Firebase Firestore
- React Router DOM
- CSS

---

## Installation

```bash
git clone https://github.com/yourusername/movie-watchlist.git

cd movie-watchlist

npm install
```

---

## Create .env

```env
VITE_FIREBASE_API_KEY=

VITE_FIREBASE_AUTH_DOMAIN=

VITE_FIREBASE_PROJECT_ID=

VITE_FIREBASE_STORAGE_BUCKET=

VITE_FIREBASE_MESSAGING_SENDER_ID=

VITE_FIREBASE_APP_ID=
```

---

## Start

```bash
npm run dev
```

---

## Build

```bash
npm run build
```

## Preview
